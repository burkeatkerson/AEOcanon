AEO Canon brand kit

◆ How to use this page — read first

This is the single source of truth for the AEO Canon identity. When you (designer or developer) build any page, email, slide, or asset for this brand, pull the logo, colors, and type from here — do not redraw or recolor them.

Logo: use the primary horizontal lockup (§2) in nav bars and headers. Use the symbol alone (§4) for favicons, app icons, avatars, and tight spaces. Use the stacked lockup (§3) for covers, cards, and centered layouts. Always keep clearspace (§6) and respect min sizes.

Backgrounds: on light surfaces use the default mark; on dark or photographic surfaces use the on-ink / reversed mark (§5). Never place the colored mark on a busy background.

Color & type: use only the tokens in §7 and the families in §8 — they match the live site (--accent, --ink, Newsreader / Spline Sans / JetBrains Mono). Copy-paste the ready snippets in §10 directly into markup.

The throughline to preserve: authoritative, edited, and “the cited answer.” The descending “first-result” symbol means you are what the engine recommends first — keep that meaning intact.

08

Typography

Three families, one job each. Newsreader = editorial authority (headlines, the wordmark). Spline Sans = clear UI & body. JetBrains Mono = labels, metadata, code.

Copy-paste assets

Drop these straight into site markup. The wordmark is text (no image needed); the symbol is inline SVG. Both inherit theme via CSS variables.

<!-- wordmark · text -->
<span class="wm">
  <span class="aeo">AEO</span>
  <span class="can">Canon</span>
</span>
/* wordmark css */
.wm{font-family:var(--serif);display:inline-flex;align-items:baseline}
.wm .aeo{font-weight:600}
.wm .can{font-style:italic;color:var(--accent);margin-left:.26em}
<!-- symbol · inline svg (first-result mark) -->
